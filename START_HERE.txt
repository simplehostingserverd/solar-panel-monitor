⚡ QUICK START (if you want to dive right in):

1. Generate a secret:
   openssl rand -base64 32

2. Edit .env file and add:
   - The secret you just generated (NEXTAUTH_SECRET)
   - Your Enphase Site ID (find it in Enphase Enlighten dashboard)

3. Run:
   ./start-local.sh

4. Open:
   http://localhost:3000

🔧 WHAT YOU NEED TO DO:

  1. Get your Enphase Site ID
     • Go to: https://enlighten.enphaseenergy.com/
     • Find it in the URL after clicking your system

  2. Generate NEXTAUTH_SECRET
     • Run: openssl rand -base64 32
     • Copy the output

  3. Update .env file
     • Add your Site ID
     • Add your generated secret

  4. Start the app
     • Run: ./start-local.sh


❓ NEED HELP?

  • Can't find Site ID? Check GET_STARTED.md
  • Docker issues? Make sure Docker Desktop is running
  • Authentication problems? Check .env configuration
  • Still stuck? Read README.md for detailed troubleshooting
